# Command: Check Real App

## ใช้เมื่อ
ต้องการ check real app ตามมาตรฐาน BDA

## Copy this into AI

```text
ทำงาน: Check Real App
Context: <วาง task/ไฟล์/ลิงก์/ข้อจำกัด>
โปรดทำตามขั้นตอนนี้:
1. เปิด/รันแอปตามวิธีของ repo
2. เช็กหน้าหรือ endpoint จริง
3. ตรวจ console/log
4. capture evidence ถ้าทำได้

Output ที่ต้องส่ง: real app check result, URL/command/log
```

## Checklist
- [ ] เปิด/รันแอปตามวิธีของ repo
- [ ] เช็กหน้าหรือ endpoint จริง
- [ ] ตรวจ console/log
- [ ] capture evidence ถ้าทำได้
