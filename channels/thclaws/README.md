# ช่องทาง thClaws — คู่มือกล่องส่วนตัวบน GX10

thClaws คือกล่อง (container) ส่วนตัวของพนักงานแต่ละคนบนเซิร์ฟเวอร์ GX10 ใช้งานผ่าน web UI ในกล่องมี AI agent (codex) + เชลล์จริง + docker ครบ — แต่**เครื่องมือฝั่ง llm-local ไม่มีในกล่องนี้** (ดูตารางท้ายหน้า)

ถ้าคุณใช้ Hermes / Claude Code / Cline บนเครื่องตัวเอง ให้ดู `../llm-local/README.md` แทน

## 1. แท็บ Shell vs แท็บ Terminal — คนละอย่างกัน อย่าหลง

| แท็บ | คืออะไร | พิมพ์อะไรใส่ |
| --- | --- | --- |
| **Terminal** | **แชทสั่ง AI (codex)** — ไม่ใช่เชลล์ | ภาษาคน: "ช่วยแก้บั๊ก...", "รัน test ให้หน่อย" |
| **Shell** | เชลล์จริง (bash) | คำสั่งจริง: `ls`, `git status`, `docker ps` |

อาการหลงแท็บที่พบบ่อย และวิธีสังเกต:

- พิมพ์ `ls` หรือ `cd ..` ในแท็บ Terminal แล้วได้ "คำอธิบาย" ยาว ๆ กลับมาแทนผลลัพธ์ทันที → คุณกำลังคุยกับ AI อยู่ ไม่ใช่เชลล์ ให้สลับไปแท็บ Shell
- พิมพ์ `/bda-repo` หรือ slash command อื่นแล้วไม่มีอะไรเกิดขึ้น/AI งง → กล่องนี้**ไม่มี slash commands** (เคยมีคนติดอยู่ตรงนี้มาแล้วจริง ๆ) ให้พิมพ์สั่งเป็นภาษาคนธรรมดาแทน
- พิมพ์ประโยคภาษาคนในแท็บ Shell แล้วเจอ `command not found` → นั่นคือเชลล์จริง ให้สลับกลับมาแท็บ Terminal
- จุดสังเกตเร็วที่สุด: เชลล์จริงตอบทันทีและสั้น ส่วน AI ใช้เวลาคิดและตอบเป็นย่อหน้า

## 2. docker / docker compose ในกล่อง

กล่องนี้เป็น sysbox container: **ใช้ docker และ docker compose ในกล่องได้เต็มรูปแบบ** ปลอดภัย ไม่กระทบเครื่อง host และไม่ต้องขอสิทธิ์พิเศษ

```bash
docker run --rm hello-world
docker compose up -d
```

สร้าง/ลบ container ในกล่องตัวเองได้ตามสบาย — ทุกอย่างถูกจำกัดอยู่ในกล่องของคุณคนเดียว

## 3. Preview — แอปต้องฟัง port 3000

แท็บ Preview จะแสดงเฉพาะแอปที่ฟังอยู่บน **port 3000** เท่านั้น ถ้าเปิด Preview แล้วว่างเปล่า ให้เช็คก่อนว่าแอปรันบน port 3000 จริงไหม

ตัวอย่าง:

```bash
npm run dev -- --port 3000
python -m http.server 3000
docker run --rm -p 3000:80 nginx
```

## 4. Whisper — ถอดเสียงเป็นข้อความ (ภาษาไทยได้)

1. อัปโหลดไฟล์เสียงผ่านหน้า upload ของกล่อง (ขนาดไม่เกิน **200MB**)
2. ไปที่แท็บ Terminal (แชท AI) แล้วสั่งให้ AI ถอดความ เช่น "ถอดเสียงไฟล์ meeting.m4a ที่เพิ่งอัปโหลด เป็นภาษาไทย แล้วสรุปประเด็น"

ภาษาไทยถอดได้ (ระบบใช้ Whisper large-v3) — ไฟล์ใหญ่กว่า 200MB ให้ตัดแบ่งก่อนอัปโหลด

## 5. Clone repo — https + token เท่านั้น

ในกล่อง**ไม่มี ssh key** — clone ผ่าน https พร้อม token เท่านั้น:

```bash
git clone https://<token>@github.com/BigDataAgency/<repo>.git
```

- ขอ token (fine-grained, สิทธิ์เท่าที่จำเป็น) จาก GitHub ของตัวเองหรือจาก admin ตาม SOP
- อย่าฝัง token ลงไฟล์ในโปรเจกต์หรือ commit ลง repo — ใช้เฉพาะตอน clone/pull/push
- `git clone git@github.com:...` (แบบ ssh) จะไม่ทำงานในกล่องนี้

## 6. สิ่งที่ "ไม่มี" ในกล่อง

ของต่อไปนี้เป็นเครื่องมือฝั่ง llm-local — อยู่บนเครื่องพนักงาน ไม่ได้อยู่ในกล่อง อย่าเสียเวลาหา/เรียกในกล่อง:

| ไม่มีในกล่อง | แล้วใช้ที่ไหน/ทำยังไงแทน |
| --- | --- |
| `aks` / `bda` CLI (`bda start`, `bda update`, ...) | ใช้บนเครื่องตัวเองผ่านช่องทาง llm-local (`../llm-local/README.md`) |
| Hermes | ใช้บนเครื่องตัวเอง (llm-local) |
| Cline / IDE extension | ใช้บนเครื่องตัวเอง (llm-local) |
| `/slash commands` (เช่น `/bda-repo`, `/fix-bug`) | ในกล่องพิมพ์สั่ง AI เป็นภาษาคนในแท็บ Terminal แทน; slash commands ใช้ได้เฉพาะ Claude Code interactive ฝั่ง llm-local |
| Obsidian vault | ใช้บนเครื่องตัวเอง (llm-local); ในกล่องให้จดเป็นไฟล์ markdown ในโปรเจกต์แล้วค่อยย้ายออก |

ส่วนที่ใช้ได้**ทุกช่องทาง**คือ `core/` — workflow, policy, checklist, template และเนื้อหา command กลางทั้งหมดใน `../../core/` ใช้กับงานในกล่องได้ปกติ (แค่ไม่ได้เรียกผ่าน slash command — ให้เปิดไฟล์อ่านหรือบอก AI ให้ทำตามเนื้อหาไฟล์นั้นแทน)
