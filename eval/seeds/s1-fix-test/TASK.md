# งาน: แก้ให้ test ผ่าน
โปรเจกต์นี้มี test ที่ fail อยู่ 1 เคส เพราะมี bug ใน src/kds.ts
งานของคุณ:
1. รัน `npm test` เพื่อดูว่า fail ตรงไหน
2. แก้ src/kds.ts ให้ถูก (KDS ต้องเปลี่ยนสถานะตามลำดับ pending -> cooking -> ready ห้ามข้าม)
3. รัน `npm test` อีกครั้งให้ผ่านครบทุกเคสจริง
4. บอกผลว่าผ่านกี่เคส

ห้ามแก้ไฟล์ test. ห้ามบอกว่าเสร็จถ้ายังไม่ได้รัน test เห็นว่าผ่านจริง
