## วิธีวัด (กันสรุปผิด)

2. **verify อิสระ** — clone/รัน test เอง + เช็ค git log ไม่เชื่อคำ agent
3. **verify ตัว verifier** — scoreboard แรกบอก scaffold overclaim 3/3 ผิด เพราะ harness หา package.json แค่ root แต่ agent สร้างใน subdir → ใช้ `reverify_scaffold.py` (recursive) + spot-check ด้วยตา
4. **โจทย์ discriminating** — S1 "แก้ให้ test ที่ fail ผ่าน" วัด self-verification ตรงกว่า scaffold
